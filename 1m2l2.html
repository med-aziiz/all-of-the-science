Lesson 2.2: Functions, Relations & Sequences

This lesson highlights the dynamic side of mathematics. We’ll explore how functions, relations, and sequences describe the way elements interact and change over time.

Functions: The Mathematical Mappings

Definition: A function is like a vending machine: you provide an input (money) and receive an output (snack) according to a set rule.

  • Domain: The set of all possible inputs.
  • Range: The set of all possible outputs.
  • Notation: f(x) represents the function f acting on an element x.

Example: Consider f(x) = 2x + 3. If x = 2, then f(2) = 2(2) + 3 = 7.

Relations: Connecting the Dots

Definition: A relation is a set of ordered pairs that shows how elements from one set correspond to elements in another.

Example: Suppose R = {(1, 'one'), (2, 'two'), (3, 'three')}; here, the number 1 is paired with the word "one," and so on.

Types of Relations:

  • Equivalence Relations: Relations that are reflexive, symmetric, and transitive—creating natural groupings.
  • Ordering Relations: Indicate how elements are arranged in a specific order (e.g., smallest to largest).

Sequences: Ordered Lists of Elements

Definition: A sequence is an ordered list of elements that follow a particular rule.

  • Arithmetic Sequences: Each term is obtained by adding a fixed number to the previous term. Example: 3, 7, 11, 15, … (each term increases by 4).
  • Geometric Sequences: Each term is obtained by multiplying the previous term by a fixed number. Example: 2, 6, 18, 54, … (each term is multiplied by 3).

Introducing a New Concept: Series

Series: The sum of the terms of a sequence. This concept helps us understand accumulation, much like how savings grow over time.

Example: The sum of the first four terms of the arithmetic sequence above: 3 + 7 + 11 + 15 = 36.

Why These Concepts Matter

Understanding functions, relations, and sequences is essential across various fields—from engineering to economics—as they model real-world phenomena such as population growth, financial trends, and even the spread of information online.

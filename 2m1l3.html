Module 1: Introduction to Computing

Lesson 1.3: Fundamentals of Programming

This lesson introduces the basics of programming—what it is, key concepts, and a simple example using Python.

Step 1: What Is Programming?

Definition: Programming is writing instructions (code) that a computer executes to perform tasks.

Importance: It is the foundation of software development, enabling everything from small scripts to large applications.

Step 2: Understanding Programming Languages

A programming language is a formal language with a set of instructions. Popular choices include Python, Java, C++, and JavaScript. Beginners often start with Python for its simple syntax.

Step 3: Basic Concepts in Programming

Variables: Containers to store data (e.g., age = 25).

Data Types: Define the kind of data (integer, string, boolean, etc.).

Control Structures: Include conditionals (if-else) and loops (for, while) to direct program flow.

Functions: Reusable blocks of code that perform specific tasks.

Example in Python:

age = 25
if age > 18:
    print("You are an adult.")
else:
    print("You are a minor.")

def greet(name):
    return "Hello, " + name + "!"

print(greet("Alice"))

Step 4: Writing Your First Program

  1. Open your code editor: Use VSCode or an online compiler (e.g., repl.it).
  2. Write a simple program: For example, to print "Hello, World!":
print("Hello, World!")
  1. Save and run the program: Observe the output on your screen.

Experiment: Modify the program to ask for the user’s name and then greet them.

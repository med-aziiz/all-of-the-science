Lesson 1.2: Propositional & Predicate Logic

This lesson introduces the formal structure of logic—think of it as learning the grammar of sound reasoning. We’ll examine two key branches: propositional logic and predicate logic.

Propositional Logic

Definition: It deals with simple statements (propositions) that are either true or false.

Basic Building Blocks:

  • Proposition: A statement like “It is raining.”
  • Connectives: Logical operators such as “and”, “or”, “not”, “if…then” that combine simple propositions into more complex ones.

How It Works

Consider two propositions:

  • Proposition A: “It is raining.”
  • Proposition B: “The ground is wet.”

Using “if…then”, you form the statement: “If it is raining, then the ground is wet.” A truth table can test all possible scenarios.

Key Inference Rules

  • Modus Ponens: If “If A then B” is true and A is true, then B must be true.
  • Modus Tollens: If “If A then B” is true and B is false, then A must be false.

Predicate Logic

Predicate logic adds detail by incorporating predicates and quantifiers.

  • Predicate: A statement containing variables, such as “is even,” which applies to different numbers.
  • Quantifiers: Words like “all” and “some” that indicate the scope of the predicate.

Universal Quantifier (∀): Means “for all” (e.g., ∀x, if x is a bird then x can fly—even though some birds, like ostriches, cannot fly).

Existential Quantifier (∃): Means “there exists” (e.g., ∃x such that x is a prime number greater than 2 and even—which applies only to the number 2).

Why Use These Systems?

  • Clarity: They allow us to express complex ideas in precise, unambiguous terms.
  • Analysis: They serve as robust tools for evaluating arguments.
  • Application: These systems underpin fields such as computer science, mathematics, and linguistics.

Example

Consider the statement: “Every student in the class passed the exam.” In predicate logic, this can be represented as:

Let P(x) denote “x passed the exam.”

Then, the statement becomes: ∀x (Student(x) → P(x)).

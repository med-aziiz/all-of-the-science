Lesson 3.1: Calculus & Analysis

This lesson covers the fundamental ideas of calculus and analysis. We’ll take a step-by-step approach to understand limits, continuity, derivatives, integrals, and convergence.

Step 1: Understanding Limits

What is a Limit?

Definition: The limit of a function f(x) as x approaches a value c is the value that f(x) gets arbitrarily close to—even if f(c) is not defined. We write this as: lim(x→c) f(x) = L.

A Detailed Example: Factoring to Find a Limit

Consider the function: f(x) = (x² - 1) / (x - 1). Notice that when x = 1 the function is undefined (denominator becomes 0). However, by factoring:

  1. Recognize that x² - 1 is a difference of two squares. (Recall: a² - b² = (a - b)(a + b).)
  2. Here, a = x and b = 1, so x² - 1 = (x - 1)(x + 1).
  3. Simplify: f(x) = [(x - 1)(x + 1)]/(x - 1) = x + 1 for x ≠ 1.
  4. Then, lim(x→1) f(x) = 1 + 1 = 2.

Step 2: Establishing Continuity

What is Continuity?

A function f(x) is continuous at a point c if:

  • f(c) is defined.
  • lim(x→c) f(x) exists.
  • lim(x→c) f(x) = f(c).

Everyday Analogy: Drawing a continuous curve without lifting your pencil.

Step 3: Derivatives – Measuring Instantaneous Change

What is a Derivative?

The derivative of a function f(x) at a point x is given by: f'(x) = lim(h→0) [f(x+h) - f(x)]/h. This represents the slope of the tangent line at that point.

Example with f(x) = x²:

  1. Set up the difference quotient: [f(x+h) - f(x)]/h = [(x+h)² - x²] / h.
  2. Expand: (x+h)² = x² + 2xh + h².
  3. Simplify: (2xh + h²)/h = 2x + h (for h ≠ 0).
  4. Take the limit as h → 0: f'(x) = 2x.

Step 4: Integrals – Summing Infinitesimal Pieces

What is an Integral?

A definite integral computes the net area under the curve of f(x) from x = a to x = b: ∫[a to b] f(x) dx.

The Fundamental Theorem of Calculus connects differentiation and integration—differentiating an integral returns the original function (with the appropriate conditions).

Example: Find the area under f(x) = x from x = 0 to x = 3:

  1. Set up the integral: ∫[0 to 3] x dx.
  2. Find the antiderivative: x²/2.
  3. Evaluate: [x²/2] from 0 to 3 = (9/2) - 0 = 4.5.

Step 5: Convergence – Understanding Infinite Processes

What is Convergence?

A sequence or series converges if its terms approach a specific finite value as the process continues indefinitely.

Example: Infinite Geometric Series

Consider the series: 1/2 + 1/4 + 1/8 + …. Written in summation form: S = Σ (1/2)ⁿ for n=1 to ∞. Using the formula for an infinite geometric series (S = a/(1 - r) with a = 1/2 and r = 1/2), we get: S = (1/2)/(1 - 1/2) = 1.

Lesson 1.2: Basic Hardware & Binary Systems

Learn about the physical components of a computer and the binary system—the fundamental language of computing.

Step 1: Understanding Computer Hardware

What Is Hardware?

Hardware refers to the tangible components of a computer. Key parts include:

  • Central Processing Unit (CPU): The "brain" that executes instructions.
  • Memory (RAM): Temporary storage for active tasks.
  • Storage (Hard Drives, SSDs): Permanent data storage.
  • Input/Output Devices: Tools like keyboards, mice, monitors, and printers.

Step 2: The Binary System

What Is Binary?

Binary is a number system using only two digits: 0 and 1. Computers use binary because their electronic components have two states: off (0) and on (1).

Step 3: How Binary Works

Key Concepts:

  • Bit: The smallest unit of data (0 or 1).
  • Byte: A group of 8 bits (e.g., 01001010).

Converting Binary to Decimal:

  1. Consider the binary number 1011.
  2. Place values from right to left are 1, 2, 4, 8, …
  3. Calculation:
    • 1 × 8 = 8
    • 0 × 4 = 0
    • 1 × 2 = 2
    • 1 × 1 = 1
  4. Sum: 8 + 0 + 2 + 1 = 11

Thus, 1011 in binary equals 11 in decimal.

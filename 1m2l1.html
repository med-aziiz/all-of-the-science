Lesson 2.1: Set Theory & Mathematical Notation

Set theory is often called the language of mathematics because it introduces a system to group objects into collections known as sets. This foundational concept paves the way for discussing more advanced topics later on.

What is a Set?

Definition: A set is simply a collection of distinct objects (elements). Think of it as a basket that holds fruits—each fruit is an element, and the basket is the set.

Notation

  • Curly Braces: Used to denote a set. For example, {apple, banana, cherry} represents a set of fruits.
  • Element Symbol (∈): Indicates membership. For instance, apple ∈ {apple, banana, cherry}.

Basic Operations and Notation

  • Union ( ∪ ): Combines all the elements from two sets. Example: If A = {1, 2, 3} and B = {3, 4, 5}, then A ∪ B = {1, 2, 3, 4, 5}.
  • Intersection ( ∩ ): Takes only the common elements between two sets. Example: A ∩ B = {3} in the above case.
  • Complement: Consists of all elements not in a given set, relative to a universal set.
  • Subset ( ⊆ ): Set A is a subset of B if every element in A is also in B. Example: {1, 2} ⊆ {1, 2, 3}.
  • Power Set: The set of all possible subsets of a set. Example: The power set of {1, 2} is {∅, {1}, {2}, {1, 2}}.

Why It Matters

Set theory provides the vocabulary for nearly every branch of mathematics. It offers a precise way to talk about collections and is fundamental to understanding functions, relations, and even probability.

Introducing a New Concept: Cardinality

Cardinality: Refers to the number of elements in a set. For finite sets, it’s a simple count; for infinite sets, it helps us understand different "sizes" of infinity—a magical concept that underpins higher mathematics.

Fun Example

Imagine you have a set of your favorite snacks: S = {chips, cookies, fruit}. Although the union of S with another set of drinks D = {soda, water, juice} isn’t directly defined, you could form a larger set of party items P = S ∪ D = {chips, cookies, fruit, soda, water, juice}. Simple, yet essential!

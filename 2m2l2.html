Lesson 2.2: Algorithm Design & Complexity

An algorithm is a step-by-step procedure to solve a problem. This lesson explains how to design algorithms and analyze their efficiency.

What Is an Algorithm?

Definition: A finite, step-by-step procedure to solve a specific problem or perform a task.

Analogy: A recipe for baking a cake.

Designing an Algorithm: A Detailed Walkthrough

  1. Understand the Problem: Restate it in your own words; identify input and desired output.
    Example: “Find the largest number in a list.”
  2. Break Down the Problem: Divide the problem into smaller steps and write pseudocode.
    Pseudocode Example: 
    function findMax(list):
    max = list[0]
    for each element in list:
        if element > max:
            max = element
    return max
  3. Analyze Efficiency:
    • Time Complexity: How the running time grows (e.g., O(n)).
    • Space Complexity: How much extra memory is needed (e.g., O(1)).
